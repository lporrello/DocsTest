Create an Asset

  1. From the , click Content.
  2. From the header, click Create > New Asset.
  3. In the New Asset screen, select a location for the asset; click Select a folder and pick one from the list. (If you have gone into an existing folder before clicking Create Assets, that folder is selected by default.)
  4. In the Image field, click Choose a file from your computer to browse for and select the image. File size limit is 3 MB. Supported file types are JPG, GIF, PNG.
  5. In the Text field, provide message content. Note the character counter in the lower right-hand corner.
  6. In the Properties pane, Asset Name field, add an internal title for the asset; this Title will be used in the list view to help users find it.
  7. In the Asset Description field, add text describing the image or message.
  8. In the Labels field, choose labels that will apply to the outbound message.
  9. In the Start Date field, if the asset is to be quarantined until a certain future date, provide the date. If the asset can be used immediately, leave it blank; defaults to current date.
  10. In the End Date field, set the date at which asset must no longer be used.
  11. In the Suitable For field, select one or more Social Channels that the asset can be used. (For example, longer text messages may not be suitable for Twitter, but could be used on Facebook or LinkedIn.
  12. In the Intended Use field, select Publication and Moderation, Publication, or Moderation. Publication means the asset is appropriate to post to one of the social channels. Moderation means the asset is appropriate to use to respond to items in the Inbox. This categorization helps users find an appropriate asset in the Content Center, but it is not restrictive; you can use Publication assets from the Inbox, or publish Moderation assets.
  13. Click Create Asset.
