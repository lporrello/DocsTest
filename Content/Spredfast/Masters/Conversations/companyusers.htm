Add and Manage Users

Add users, assign/reassign Roles, Workspaces, Initiatives, Account Sets, and Approval teams. Export a list of users. Change the Owner of the Company.

Add a New User to the System

  1. From [your name] > My Company, click the Users tab.
  2. Click Add User.
  3. Provide first name, last name, and email address. The address provided in the Email Address field will be used to create the username. (You can add a second email address to contact the person later.)
  4. Select User Role from the drop-down menu.
    For more information about roles and permissions, refer to the User Guide.For more information, see User Roles and Permissions
  5. In the Workspace menu, select a Workspace for the user.
    For more information about Workspaces, refer to the User Guide.For more information, see Workspaces.
  6. In the Approval Teams menu, choose individual message Approval Teams for the user, No Approval Teams or All Approval Teams.
    For more information, see Approvals.
  7. Typically, users are assigned only to individual Initiatives. In the Initiatives menu, select No Initiatives, All (current) Initiatives, or specific Initiatives for the user. If you want the user to be automatically added to all Initiatives created in the future, click the Add user to new Initiatives checkbox beneath the drop-down menu.
    For more information about Initiatives, refer to the User Guide.For more information, see Initiatives.

If you want the user to be automatically added to all Account Sets created in the future, click the Add user to new Account Sets checkbox.
    For more information about Account Sets, refer to the User Guide.For more information, see Accounts.

In the Send Invitation section, Email Address field, you can enter a second email address to send the welcome message to (the message will be sent to both addresses; the username will be created using the address in the first Email address field). If you leave this second Email Address field blank, the email invitation is sent to the address in the first Email Address field.

Reassign a User to Different Roles, Workspaces, Initiatives, Account Sets, and Approval Teams

You do not need to re-send an invitation to the user

  1. From [your name] > My Company, click the Users tab.
  2. Locate the user in the list, and click the pen icon (Edit User).
  3. From the User Role drop-down menu, select the new Role.
  4. From the Workspace drop-down menu, select the new Workspace.
  5. From the Approval Teams drop-down menu, select the new Approval Teams.
  6. From the Initiatives drop-down menu, select the new Initiative(s).
  7. From the Account Sets drop-down menu, select the new Account Sets.
  8. Click Save Changes.

For information about changing users’ email and passwords, see Password and Account Security.

Change a User’s Email (if permitted) and Password

  1. From [your name] > My Company, click the Users tab.
  2. Locate the user in the list, and click the pen icon (Edit User).
  3. (If permitted) Click Change Email.
  4. Enter your password.
  5. Provide the new email address.
  6. Click Save.
  7. Click Change password.
  8. Enter your password.
  9. Provide and Confirm the new password.
  10. Click Save.

Export a CSV File With a List of Users

The list will contain the following information.

  • First name, Last name
  • Email address
  • Role
  1. From [your name] > My Company, click the Users tab.
  2. Click Export.
  3. Choose to open or save the file.
  4. Click Ok.

Remove a User

  1. From [your name] > My Company, click the Users tab.
  2. Find the user in the list, and click the trash icon (Remove User).
  3. In the Remove User dialog, click Remove to confirm the deletion.
    For information on changing users’ Workspace, see Workspaces.
  4. If you are the user identified as the “Company Owner” in , the Change Owner button on this page becomes active, allowing you to choose a new owner.

Change the Company Owner

  1. From [your name] > My Company, click the Users tab.
  2. In the table, find the user who will be the new owner.
  3. Click the checkbox to the left of the person’s name.
  4. Click Change Owner.
