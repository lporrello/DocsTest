Import Assets

The following rules pertain to the Asset Import tool:

  • You must upload assets into a specific, named folder (not the All Assets view).
  • Assets are uploaded using a CSV file. All columns of the CSV are optional.
  • You can import up to 200 assets at a time; each row in the CSV contains one asset.
  • Image files are uploaded individually after the CSV file is uploaded; take time to organize your image files so you can easily associate each file with the appropriate row in the CSV file.

Import Multiple Assets to the Content Center:

  1. In the Management bar of the Content Center, click Import Assets (cloud) icon.
  2. Select a folder (other than the All Assets folder) and click Choose.
  3. Click Download CSV template, open it using Excel or another spreadsheet or text editor, and complete a row for each asset. Remember that image files will be uploaded separately. For each image asset, complete the text and properties fields.
  4. Click Upload New File. The messages are evaluated by .
  5. On the Import Assets page, review the messages. Ones that are incorrectly formatted or have other errors are identified.
  6. Correct Invalid Entries Before Importing
  7. If validation errors are found, we recommend you cancel the import, fix the errors in the CSV, then import the corrected CSV file into . If you import a partial CSV file (import just the valid entries), then correct the errors in the CSV file and re-import it, the valid assets that have already been imported will be duplicated in .
  8. If the asset includes an image, click Add Image within the row, and upload the file.
  9. Click Import [#] Entries to finish uploading them in to .
