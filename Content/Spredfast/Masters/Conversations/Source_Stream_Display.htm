Source Stream Display

In streams that contain content from many other streams (such as a Collection, the Primary Collection, the Highlighted stream, or the Hidden/Deleted stream), the name of the source stream in which the item initially appeared is displayed on the item, preceded by an icon indicating the type of source stream.

If a stream is renamed, the new name appears on all items received after the change was made. Items received before the change retain the original name. If the item could have originated from several different streams, uses the stream in which it appeared first (earliest).

When a user responds to the item, defaults to the publishing account associated with the source stream that first received the item. If you want to publish using a different account, you can change it in the publishing dialog. Note that a default account is only used for account-based streams; it is not selected for search-based streams.
