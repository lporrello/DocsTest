Change Time Display on Stream Items

A setting in the Stream Library menu allows stream items to show the actual date and time an item appeared on the social channel, instead of the relative time (“X minutes ago”) shown by default for all items less than one hour old. The setting is on a per-user basis, and applies to all Initiatives the user has access to.

Show Absolute Time On All Stream

From the Stream Library menu, change the Absolute Stream Time option to ON.
