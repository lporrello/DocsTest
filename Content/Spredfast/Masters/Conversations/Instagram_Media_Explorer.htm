Instagram Media Explorer

For details about filters on Media Explorer, see Company-wide Analytics Filters.

In the Media Explorer, the Total Media number is a drill-down; when you click it, additional columns of message-related data are provided.

Instagram Media Explorer Columns

Metric

Definition

Date

Date the message was published.

Time

Time the message was published, in Pacific Time.

Account

The account used to publish the message.

Labels

Any outbound label(s) applied to the message.

Image

Click the image thumbnail to open external link in a new window

Text

The text (description) for the media. Click on the text link to access Message Details.

Channel ID

Unique ID for the message, created by the channel. Click the link to see the item on the native channel, in new window. (You must be logged in to the channel.)

Engagement

Sum of likes and comments the media received.

Likes

Number of Likes s the message received.

Comments

Number of comments the message received.

 

Instagram Additional Columns, from Total Media

Column

Definition

ID

Internal message identifier, used to de-duplicate data.

Initiative Name

The name of the Initiative in which the message was published.

Account

The Account used to publish the message.

User Name

The name of the () user who published the message. If the message was auto-imported, (Auto import) appears in the column.

Title

The (internal, non-published) title of the message.

Text

The descriptive text for the message.

Published Media

Internal value, used for calculation purposes.
