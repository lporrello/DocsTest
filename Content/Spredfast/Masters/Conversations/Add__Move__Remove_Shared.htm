Add, Move, Remove Shared Bookmark

The shared bookmark helps users who work together in the Inbox manage transitions during shift changes or breaks in the workday. A user can mark a single item in each stream or shared collection with a bookmark icon that others in the Initiative can see. As shifts change or people resume work after a break, the stream location where work stopped can be easily identified. Following are some shared bookmark feature rules:

  • One bookmark is available for each shared collection or stream (except for the Archived view of the Primary Collection stream and the Highlighted stream, which is personal, not shared).
  • All users in the Initiative can see the shared bookmark.
  • Any user in the Initiative can add the bookmark to an item, move it to a different item, or delete it from an item.
  • In the Primary Collection stream, if the item with the bookmark gets archived by a user, automatically moves the bookmark to the next item (more recent) in the Primary Collection stream.
  • Once a bookmark has been added to an item in the stream, the Jump to menu at the top of the stream includes a “shared bookmark” option, so that any user in the Initiative can go directly to the bookmarked item.
  • Deleting a stream or shared collection that has a bookmark applied in it deletes the bookmark.
  • If the stream appears in multiple tabs, the bookmark’s location is maintained in all instances of the stream.
  • If an applied filter causes the bookmarked item to be hidden (either because of a user-applied filter or the read/unread filter in the Primary Collection), the filter must be removed before the user can Jump to the bookmark.
  • Hover over a bookmark to see who applied it, and when it was added or moved.

Use a Shared Bookmark in the Inbox

  1. From within a stream item, click the More actions menu, and select Add shared bookmark. The bookmark icon appears in the upper right corner of the item.
  2. Once the bookmark has been set, any user in the Initiative can use the Jump to menu to find the item. Select Jump to > Shared bookmark.
  3. Any user in the Initiative can move the bookmark to a different item within the stream, or delete it, using the More actions menu.
