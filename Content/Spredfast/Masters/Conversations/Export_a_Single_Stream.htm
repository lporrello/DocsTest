Export a Single Stream

This feature exports the most recent 5000 items in a stream. To export data from multiple streams, filter the stream content before you export it, or export stream activity, see Export Stream Content or Activity.

  1. Click the Stream Management (chevron) icon in the stream header
  2. Choose Export Stream. A message confirms that the file has been exported.

Exported fields include the following:

  • stream name
  • ID
  • Title
  • Text
  • Author user name
  • Author User ID
  • Author link
  • Author icon
  • Link to the item
  • Date
  • Labels

For Bazaarvoice streams only...

  • Positive Feedback Count
  • Negative Feedback Count

For Brandwatch streams only...

  • Sentiment
  • Impact
  • Tags
  • Source
