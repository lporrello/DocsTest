Microblog Series

The Series feature allows you to create empty “shells” of a single (microblog) message, schedule them, and assign others to write content for them. It’s an easy way to set up messages that will have different content on a regular schedule.

Difference between Microblog Series and Repost
“Repost” sends the same (identical) message multiple times. Microblog “Series” sends different content on a repeating schedule.

Create a Microblog Message Series

  1. From the > Publish menu, click the circled ... (ellipsis) icon to the right of the publishing icons.The Schedule Message screen appears.
  2. In the Name field, give the series a name (the name is internal-only, and will not be published).
  3. Select Facebook Series, Google+ Series, LinkedIn Series, Weibo Series, or Twitter Series.
  4. In the Date field, use the calendar to select the date on which the first message in the series will publish.
  5. In the Time field, use the picker to set the time the messages will publish
  6. In the Repeat field, choose the increment at which messages will repeat: Daily (by the day), Weekly (by the week) Monthly by Day of Month (May 9, June 9, July 9) or Monthly by Day of Week (Second Wednesday in May, Second Wednesday in June, Second Wednesday in July).
  7. In the Repeat every drop-down, select the frequency at which the message should repeat.
  8. In the End Date field, use the calendar to choose the date to stop the series.
  9. In the Label field, add labels to the posts.
  10. In the Assign to field, select the assignee. If you want to use different assignees for each message in the series, you can change the assignee for each one by editing the individual instance of the message from the agenda calendar.
  11. In the Add approval path field, If you want to use supplemental approval paths for the series (in addition to the ones required for the Account Set) select them from the list.
  12. Click Save. All messages in the series are put on the schedule and can be edited from the Calendar > Agenda entry for the item.
