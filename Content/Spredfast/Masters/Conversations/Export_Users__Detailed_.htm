Export Users (Detailed)

To export detailed user and company data to a CSV, click Admin > User Export.

Data columns include the following:

  • user id
  • First Name
  • Last Name
  • Email Address
  • last login timestamp (human readable)
  • Internal role
  • Company name (e.g., Bayer)
  • Company id
  • Company role (e.g., Company Admin)
  • Site
  • Company service level (e.g. Trial)
  • Trial expiration.

If a user is in multiple companies, he appears in multiple rows (one for each company).
