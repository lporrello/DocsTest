Process Multiple Stream Items

The multiple items processing menu at the top of each stream column enbables you to take action on multiple items in the stream.

To identify items to process: For each item, hover over the upper left-hand corner of the item. Click the check icon that appears--it turns blue when the item has been selected. To select all currently visible items, click the Select All checkbox at the top of the stream. Then click the drop-down list to display available actions. Depending on the type of stream, these can include the following:

  • applying a macro to the items
  • highlighting the items for other users (the users must have workspaces that allow access to the Inbox)
  • adding labels to the items
  • adding notes to the items
  • marking the items as read (An item is automatically marked “read” if a user performs any action on it.)
  • archiving the items (the Primary Collection stream archive is for all users in the Initiative; the Highlighted stream archive is user-specific).
  • identifying authors of the items as Influencers
  • accepting/releasing items (from the Highlighted stream)
