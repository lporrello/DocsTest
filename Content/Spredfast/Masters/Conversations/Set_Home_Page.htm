Set Home Page

You can choose one page in as your Home Page; the page will be the first that appears to you when you log in. On any page (Inbox, Calendar, etc.), move your cursor in the space to the right of the page name to display the Home Page Available icon. Click the icon to set the page as your Home Page.

Home Page
Icon Description
Home Page Available —the page can be selected as your Home Page
Home Page Set—the page has been selected as your Home Page
