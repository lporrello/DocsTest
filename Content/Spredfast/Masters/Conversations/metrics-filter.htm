Initiative Explorer Filter

Filter analytics data using these criteria. Note that all criteria might not be available within each view.

Label

Data associated with one or more labels. The Label filter affects metrics for Total Reach, Activity, Engagement and Value, providing data associated with the selected label.

Social Channel

Data associated with one or more social media channel (e.g. Facebook). The Social channel filter affects metrics for Total Reach, Activity, Engagement and Value, providing data for the accounts and content generated for the selected channel.

Content Type

Data associated with one or more content types, such as blog, comment, photo and the like. The Content type filter affects metrics for Total Reach, Activity, Engagement and Value.

Account

Data associated with one or more social media account (or “channel”) such as a single Twitter account. The Account filter affects metrics for Total Reach, Activity, Engagement and Value, providing data for the content generated by the selected account.

Account Set

Data associated with one or more Account Sets. The Account Set filter affects metrics for Total Reach, Activity, Engagement and Value, providing data for the accounts and content generated by the selected Account Set.

User

Data associated with one or more individual users. By default, all data for all users is provided. The User filter applies to data for Activity, Engagement and Value, showing activity created by the user and the engagement that activity received.

When building a filter, clicking a checkbox includes the checked item's data, and excludes data for the unchecked items. For example, in Content type, checking Blog means that data for blogs will be returned, and data for Microblogs, Photos, and Comments will not be returned. You can include multiple criteria in a single filter. Select items and click Apply.

Reuse a Filter that You Have Created

  1. Click Active Filter.
  2. Click Save as new filter, and then name the filter.

At any time, you can hover over the Active Filter button to see which filter is currently in use.
