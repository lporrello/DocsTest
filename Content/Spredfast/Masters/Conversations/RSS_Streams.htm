RSS Streams

Create streams using RSS Feeds to monitor key industry blogs. Enter the URLs, separated by commas. If you use an RSS aggregator that lets you build an OPML (Outline Processor Markup Language—an XML standard often used for webfeed lists) file, you can upload it into the system as well.

  1. From the Stream Library menu, select Create Streams.
  2. From the drop-down menu, click RSS.
  3. In the RSS Feed URLs field, enter the URLs of the blogs to include in the stream, separated by commas.
  4. Click Search. Results appear in the list below.
  5. If you want to use a webfeed list, in the Import and Upload RSS Feeds field, click Upload.
  6. Click Add to add a stream. A progress monitor launches in the lower half of the Add Streams panel. While one stream loads, you can add more.
  7. When you have added all the streams you want, click Done.
