Filter a Stream

Filter stream content using the following criteria:

  • Data (including date range)
  • Labels (items with any label, no labels or specific labels; you can use advanced expressions to find specific labels and exclude others)
  • Influencer (whether the author of the item has been identified as an Influencer)
  • Visibility (view or hide dark posts)
  • Notes (items with or without notes)
  • Search terms
  • Responses (whether users have published or have pending, a response to the items)
  • Retweets (to include/exclude retweets, or show only retweets).

Create a Stream Filter

  1. Click the chevron (Stream Management) icon in the stream header, and then choose Filter.
  2. Build the filter using the criteria described above.
  3. To make the filter available for reuse, click Save Filter As..., and name the filter. Then, click Save & Apply.
  4. If you do not wish to save the filter, click Apply without adding a filter name.
  5. When a stream is filtered, a notification appears in the stream heading, Filter On X. To remove the filter, click the X.

“No filter applied” means that the particular criterion (Notes, for example) is not considered in the filter. You can name and reuse filters.

If you have listening (from Sysomos or Synthesio) enabled for the stream or collection, an additional filter set appears. Across the top of the Filters dialog, you will have one tab for each vendor that is enabled for your company.

Brandwatch Tab Stream Filtering Criteria

  • Sentiment (Positive, Neutral, Negative)
  • Impact (a numerical value between 0 and 100)
  • Tags (terms passed through by the Brandwatch query).
  • In the Sysomos tab, filter the stream using the following criteria:
  • Sentiment (Positive, Neutral, Negative)
  • Influence (a numerical value between 1 and 10)
  • Tags (terms passed through by the Sysomos query)

Synthesio Tab Stream Filtering Criteria

  • Sentiment (Positive, Neutral, Negative)
  • Rank (a number between 1 and 10)
  • Topics

Sysomos Tab Stream Filtering Criteria

  • Sentiment (Positive, Neutral, Negative)
  • Rank (a number between 1 and 10)
  • Topics

Filtered Streams in Tabs
If you have applied a filter to an active stream, the filter remains in effect if you drag the stream to a new location in the tab, or to a different tab. If you remove the stream (so that it is no longer part of an active tab and appears only in the All Streams section of the Stream Library), the filter is removed.
