Tabs

Tabs contain streams and Collections. They are unique to a specific user (that is, only the user can name his tabs and select which streams and collections appear in them; an administrator cannot configure the tab names and contents for users in the Initiative). Your Inbox can have up to ten tabs, and each tab can contain up to ten (total) streams and/or Collections.

To add a tab to the Inbox, click the Add Tab button. Click in the Tab Header to add a name. To delete the tab, click the X icon to the right of the name. Rename a tab by clicking in the heading and typing a new name.

To reorder tabs, drag and drop them to a new location within the Tab title bar. Note that the tab order is updated in the Stream Library as well.

Click on a tab’s header to make it “ active,” that is, to bring it to the front of the screen view so that the streams it contains are displayed.
