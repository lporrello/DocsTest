Account Explorer

The All Channels Account Explorer provides audience size information, by day.

All Channels Account Explorer Columns

Column

Definition

Initiative Name

The name of the Initiative containing the account

Account Set

The Account Set containing the account

Channel

The social media channel containing the account.

Account

The account having the audience size

Metric

The type of data provided; Audience Size (The number of people (Facebook Friends/Fans, Google+ Followers, Twitter Followers, Weibo Followers, LinkedIn Connections/Followers, and YouTube Channel Subscribers) associated with the account.

Date, in format M/DD/YY

For this day, the Total audience size for the account.
