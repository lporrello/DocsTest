Twitter Users Streams

Monitor important Twitter accounts by providing a list of Twitter handles. Each stream must be associated with an authenticated Twitter account. Note that the number of Twitter Users streams allowed for each Twitter account is limited, see Info.

“Protected” Tweets
If a Twitter user has “protected” tweets on Twitter, you cannot create a Twitter Users stream for the user; the Tweets are not available to .

  1. From the Stream Library menu, select Create Streams.
  2. From the drop-down menu, select Twitter.
  3. Click the Twitter Users tab.
  4. Enter the handles to monitor, separated by commas. Search results appear in the Select from search results screen; when you click on an account, summary information appears in the adjacent panel.
    Each Twitter Users stream must be associated with an authenticated Twitter account. By default, the stream is assigned to the account that has the fewest streams already associated with it. To change the assignment, click Change account and select it from the drop-down menu. Note that the number of Twitter Users streams allowed for each Twitter account is limited, see Info and Twitter-Based Streams.
  5. Click Add to create a stream for the account.
  6. Click Done.
