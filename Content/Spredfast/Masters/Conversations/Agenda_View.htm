Agenda View

The Agenda shows all messages for a specific day (scheduled and published) from the Initiatives chosen in the Basic and Advanced filters. Items in the Agenda have both a collapsed and an expanded view. To switch between them, click the + icon that appears in the lower left-hand corner of the item. To expand all agenda items, click the Details slider to On in the upper left-hand corner of the screen.

Agenda Item: Details Off

  • Scheduled / Published time
  • Status (approval path/publishing status) icon
  • Image thumbnail
  • Real-Time Trends metrics
  • Initiative—In plain text for your current Initiative or italics for other Initiatives
  • Name—the internal name for the message)
  • First lines of content, including live links so you can test them
  • Channel
  • Assignee name, or if the item was auto-imported
  • Action icons, such as Edit, Delete Message, Copy message, Star.
  • Unpublished Page Post status. If the item was created as an Unpublished Page Post.
  • Promotion status, if the message has been promoted on Facebook. Rollover to see budget and end date information.

Agenda Item: Details On

To open the expanded view, click the Details slider to On at the top of the page, or select the plus icon at the bottom left corner of the post.

  • Scheduled/ Published time
  • Status (approval path/publishing status) icon
  • Image thumbnail
  • Real-Time Trends metrics
  • Initiative—In plain text for your current Initiative or italics for other Initiatives
  • Name—The internal name for the message
  • First lines of content, including live links so you can test them
  • Channel
  • Assignee name, or if the item was auto-imported
  • Action icons, such as Edit, Delete Message, Copy message, Star.
  • Unpublished Page Post status—If the item was created as an Unpublished Page Post.
  • Promotion status—If the message has been promoted on Facebook. Rollover to see budget and end date information.
  • Real-Time Trends—Metrics link and summary (if you have a Intelligence subscription).
  • Metrics (on published items)—Metrics shown here are aggregate metrics, by channel, for the message. For example, if you post a photo to multiple Facebook accounts, the counts are cumulative for all accounts within the channel. To see the data broken out by account, see Message Details The frequency at which this data is fetched depends on the age of the item. See Content Metrics.
  • Labels—Click Labels > Manage Labels to add labels.
  • Notes—Click Notes to read notes, and Manage Notes to write a new note.
  • Activity—Click Activity > View All Activity.
  • Account Sets—If the Account Set has an avatar, the image is included. If there are multiple avatars, the one for the first Account Set (alphabetical) appears.
  • Accounts—If the item was created as an Unpublished Page Post, then subsequently published using one or more accounts, the hover text describes the status for each.
  • Targeting—If targeting was applied to the message, the hover text provides details.
  • Series information (for Facebook, LinkedIn, Twitter Series, or other message types that use the Repeat/Repeat every fields)
  • Repost information for reposted microblogs
  • Approval Path—The list of approval teams used for the message; rollover the team name to see membership.
