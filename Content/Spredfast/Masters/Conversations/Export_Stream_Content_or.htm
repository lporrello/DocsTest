Export Stream Content or Activity

You can export stream content data for individual and multiple streams, or log data that shows activity taken on stream items within . An export is limited to 5000 items or one day, whichever is greater.

The stream content report contains: Stream type/name, Channel, ID (on service), Title, Text of the stream item (content), Author user name, Author user ID, Author link, Author icon, Link (to post), Date, Labels.

The stream activity log report contains for each piece of activity—reply, retweet, comment, delete, hide—any action taken by someone in the company within the selected date range (based on the date the item originated, most recent to oldest):

Stream item Log ID, Stream item Log Unix Timestamp, Stream Item Log Timestamp, Stream Item UID (Key), Stream Item Type, Stream Item Timestamp, Stream Item Unix TimeStamp, Stream Item title, Stream item text/content, Author Username (of original item), Initiative ID, Initiative Name, Stream ID, Channel, Stream Type, Credential ID, Credential Name, User Last Name (user in ), User First Name, Label IDs, Labels, User ID, Highlighted for User ID, Highlighted for User Last Name, Highlighted for User First Name, Highlighted for Group ID, Highlighted for Group Name, Message ID, Content Type, Content ID.

Export Stream Content or Activity Data to a CSV File

  1. From the Stream Library menu, click Export.
  2. Select one or more streams using the +/- icons.
  3. If you want to export only stream content (without filtering), click Export. If you want stream activity data (activity log) or to filter stream content data before exporting it, click Next, and choose either:
  4. Click Export the activity log for the selected streams, and provide a date range, or:
  5. Click Apply filters to the selected stream(s) and choose filter criteria for the content. If you apply a content filter, you can save it for reuse; click Save as new filter, and provide a name.
  6. Click Export.
