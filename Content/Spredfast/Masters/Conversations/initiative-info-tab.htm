Change the name of the Initiative and add a description.

If you no longer need the Initiative, you can suspend it. When an Initiative is suspended...

  • It no longer appears in the interface
  • Stream data is no longer fetched
  • Metrics are no longer collected
  • Historical data cannot be viewed in analytics

If you suspended an Initiative, and want it re-enabled, contact Support. There will be a gap in data from the time the Initiative was suspended and the time it is re-enabled.
