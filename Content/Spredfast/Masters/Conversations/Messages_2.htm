Date range reflects time of message publication
The date range defines when the message was published, not when the engagement was received. For example: you published a message a month ago. If you filter the data based on the week you published the message, the results return all engagement activity received for the message, over the entire month. This is because some messages, such as videos, can take a long time for their maximum engagement to be achieved.

No Engagement Metrics for Native Retweets
Though “native” Retweets posted through are counted as publishing activity in , engagement metrics are not available for them. (From Twitter’s perspective, a native Retweet is an additional instance of the original tweet, not a separate tweet that can have independent, measurable engagement. Any engagement on the Retweet would be credited to the original Tweet.)
So if you post a “native” Retweet using , and other people engage with the retweet, that engagement is not counted. Consequently, Average Engagement ratios may be skewed. If you must have engagement metrics on all retweets published through , limit users to Editable Retweets only.

Favorites Count Limitation
collects Favorites data for an account’s 3200 most recent tweets. Once a tweet falls outside that limit (Tweet 3201), its Favorites count is no longer updated. Over long periods of time, or for very active Twitter accounts, users may see a discrepancy between Twitter and Favorites counts.

Twitter Engagement by Type

The number is the sum of all engagement (clicks, replies, favorites, “native” and editable retweets) received by all messages published during the date range by all Twitter accounts in the Initiatives; click it to see the engagement sums per account. For each engagement type (Clicks, Replies, Favorites, Editable RT, Native RT), the top-level numbers represent the sum for all accounts; click on the number for each type to see engagement values of that type for each account.

  • Favorites—The number of favorite designations received by Tweets published through or auto-imported, by all Twitter accounts that are part of the Initiatives. This data is provided by Twitter.
  • Clicks—Number of clicks made on links embedded in tweets published via or auto-imported, by all Twitter accounts that are part of the Initiatives. This value includes clicks on links in Retweets and Replies to the original tweet. This data is provided by Bitly.
  • Replies—The number of replies received by Tweets published through or auto-imported, by all Twitter accounts that are part of the Initiatives. This data is provided by Twitter.
  • Editable Retweets—The sum of all retweets of the form RT@twitter_user that were received by tweets published by all Twitter accounts in the Initiatives. This data is calculated by .
  • Retweets—The sum of all retweets, made using the native Twitter retweet function, of content published by all Twitter accounts in the Initiatives. This data is provided by Twitter.
