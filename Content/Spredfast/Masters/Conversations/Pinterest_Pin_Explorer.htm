Pinterest Pin Explorer

For details about filters on Pin Explorer, see Company-wide Analytics Filters.

In the Pin Explorer, the Total Pins number is a drill-down; when you click it, additional columns of pin-related data are provided.

Pinterest Media Explorer Columns

Metric

Definition

Date

Date the pin was published.

Time

Time the pin was published, in Pacific Time.

Board

The Board used to publish the pin.

Labels

Any outbound label(s) applied to the pin.

Image

Click the image thumbnail to open external link in a new window

Text

The text (description) for the pin. Click on the text link to access Details.

Channel ID

Unique ID for the pin, created by the channel. Click the link to see the item on the native channel, in new window. (You must be logged in to the channel.)

Engagement

Sum of Likes and comments the pin received.

Repins

Number of Repins the pin received.

Likes

Number of Likes the pin received.

Comments

Number of comments the pin received.

 

Pinterest Additional Columns, from Total Pins

Column

Definition

ID

Internal message identifier, used to de-duplicate data.

Initiative Name

The name of the Initiative in which the pin was published.

Board

The Board used to publish the pin.

User Name

The name of the () user who published the pin. If the pin was auto-imported, (Auto import) appears in the column.

Title

The (internal, non-published) title of the pin.

Text

The descriptive text for the message.

Published Pins

Internal value, used for calculation purposes.
