Approvals

Before adding Approval Teams, make sure you understand how approvals work in . Read About Approvals.

Create and Add Users to an Approval team

  1. Select [Your name] > My Company > Approvals.
  2. Click Add Team.
  3. In the Approval Team Name field, type a name for the team.
  4. From the list of users, click on the + icon next to the name of each user who will be part of the team. You can use the Add All or Remove All buttons as appropriate.
  5. Click Add Team. The new team appears in the list. To edit, or remove a team, use the pen icon (Edit) icon or trash can icon (Delete).

The order in which the teams are listed on the Approvals screen is the same order that will be used for message approval, if multiple approvals are required.

Reorder Approval Teams

You can do this to change the order in which the teams will be sent a message requiring approval.

  1. Select [Your name] > My Company > Approvals.
  2. Click Rearrange. The Approval Team user lists are collapsed so that only the team names are shown. To expand the list to show the members, click the arrow to the left of the team name.
  3. Drag and drop the teams into the correct order.
  4. Click Save Order.

For more information on assigning Approval Teams to Initiatives, see Approval Setup.
