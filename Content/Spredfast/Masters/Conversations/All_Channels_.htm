All Channels

Includes data for Facebook, Twitter, LinkedIn, Instagram, Weibo, Google+, Pinterest, and YouTube.

In the Message Explorer, a message published in multiple Initiatives is de-duped by label, one row for each unique label combination (Instances of the message with no labels are in the same row.) Click on the Total Messages value (upper right) to see each instance of the message (all with the same Message ID), by Initiative, along with its labels.

For details about filters, see Company-wide Analytics Filters.

In the Message Explorers, the Total Number of Messages number is a drill-down; when you click it, additional columns of message-related data are provided.

All Channels Columns

Column

Definition

Date

Date the message was published.

Time

Time the message was published, in Pacific Time.

Channel

The social network on which the message was published.

Account

The account used to publish the message.

Labels

Any outbound label(s) applied to the message.

Image

Thumbnail of message image. Click on the link to open the image in a new window.

Text

The content of the message. Click on the text link to access Message Details.

Channel ID

Unique ID for the post, created by the channel. Click the link to see the item on the native channel, in new window. (You must be logged in to the channel.)

Engagement

Sum of engagement counts for the message, as engagement is defined for the channel, without click data.

Clicks

Number of clicks received by links embedded in the message, provided by Bitly or BudURL.

Web Conversions

If Google Analytics or Omniture has been configured for one or more of the Initiatives, the number of conversions that the message has generated.

 

All Channels Columns, from Total Number

Column

Definition

ID

Internal message identifier, used to de-duplicate message data.

Date (Message Published)

Date the message was published.

Time

Time the message was published.

Initiative Name

The name of the Initiative in which the message was published.

Account Set

The Account Set used to publish the message.

Account

The account used to publish the message.

Labels

Any outbound label(s) applied to the message.

User Name

The name of the () user who published the message. If the message was auto-imported, (Auto import) appears in the column.

Title

The (internal, non-published) title of the message.

Text

The content of the message.

# of Messages

Internal value, used for calculation purposes.

Engagement

Sum of engagement counts for the message, as engagement is defined for the channel, without click data.

FB Like Stories (Timeline)

Number of stories created by Facebook when a person liked a message.

FB Comment Stories (Timeline)

Number of stories created by Facebook when a person commented on a message.

FB Share Stories (Timeline)

Number of stories created by Facebook when a person shared a message.

FB Other Stories

Number of other stories (such as redeeming an offer or answering a poll) created by Facebook

Twitter Total Retweets

Sum of Editable and (native) Retweets

Twitter Replies

Number of replies the message received.

Twitter Favorites

Number of “Favorite” designations the message received.

Twitter Retweet Audience

The number of people who could have seen a retweet of the message, based on the Follower counts of the retweeters.

Clicks

Number of clicks received by links embedded in the message, provided by Bitly or BudURL.

Web Conversions

If Google Analytics or Omniture has been configured for one or more of the Initiatives, the number of conversions that the message has generated.

IG Comments

Number of Comments the Instagram media received.

IG Likes

Number of Likes the Instagram media received.

LI Shares

Number of shares the LinkedIn message received.

LI Likes

Number of Likes the LinkedIn message received.

LI Comments

Number of Comments the LinkedIn message received.
