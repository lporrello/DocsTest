Edit Company Name, Parent Company

If you have access to the Admin Panel, you can edit the Company Name and Parent Company. For more information about Parent Companies, see Parent Company.

Change the Company Name and Parent Company

  1. From your name > My Company, click the Info tab.
  2. In the Info field, click the Edit (pen) icon.
  3. In the Edit Company dialog, provide a new name in the Name field.
  4. In the Parent Company drop-down menu, select a new Parent Company from the list.
  5. Click Save Changes.
