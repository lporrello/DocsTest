Google+ Message Explorer

In the Message Explorers, the Total Number of Messages number is a drill-down; when you click it, additional columns of message-related data are provided.

Google+ Columns

Metric

Definition

Date

Date the message was published.

Time

Time the message was published, in Pacific Time.

Account Name

The account used to publish the message.

Labels

Any outbound label(s) applied to the message.

Image

Thumbnail of message image. Click on the link to open the image in a new window,

Text

The content of the message. Click on the text link to access Message Details.

Channel ID

Unique ID for the message, created by the channel. Click the link to see the item on the native channel, in new window. (You must be logged in to the channel.)

Engagement

Sum of clicks, +1s, comments and reshares the message received.

Clicks

Number of clicks received by links embedded in the message, provided by Bitly or BudURL (depending on which you have configured).

+1s

Number of +1s the message received.

Comments

Number of comments the message received.

Reshares

Number of reshares the message received.

Web Conversions

If Google Analytics or Omniture has been configured for one or more of the Initiatives, the number of conversions that the message has generated.

 

Additional Google+ Columns, from Total Number

Column

Definition

ID

Internal message identifier, used to de-duplicate message data.

Initiative Name

The name of the Initiative in which the message was published.

Account Set

The Account Set used to publish the message.

User Name

The name of the () user who published the message. If the message was auto-imported, (Auto import) appears in the column.

Title

The (internal, non-published) title of the message.

# of Messages

Internal value, used for calculation purposes.
