Message Import

Upload a CSV file of up to 200 messages to post to Facebook, Twitter, Google+, Weibo, or LinkedIn. You can include preview links (for channels that support them), notes, and labels along with the text of the message.

The CSV must conform to the formatting specified in the CSV template, which you can download from within the application.

Import Messages into :

  1. Create a CSV file in the required format, containing messages to be published from the same channel. To obtain a template, open the Message Importer ( > Publish > Message Importer icon), and select Download spredfast-message-importer.csv.
  2. When the CSV file is ready, click the Message Importer icon.
  3. Click Import Messages.
  4. Select a Channel from the drop-down menu.
  5. (Optional) In the Accounts pane, select the accounts (or select all accounts) through which to publish the messages. All messages in the CSV file will be scheduled to be sent the account(s) chosen from the list. If no accounts are chosen, the message is imported in Draft state.
  6. In the File area, click Browse, and select the CSV file.
  7. Click Upload. The messages are evaluated by .
  8. On the Validation page, review the messages. Ones that are incorrectly formatted or have other errors are identified.

Correct Invalid Messages Before Committing
If validation errors are found, we recommend you cancel the import, fix the errors in the CSV, then import the corrected CSV into . If you import a partial CSV (import just the valid messages), then correct the errors in the CSV and re-import it, the valid messages that have already been imported will be duplicated in .

Use the checkboxes to include messages to import. Click Commit Messages to finish uploading them in to .

From the Confirmation screen, you can view see a specific message in the Agenda (in a new tab) or Start a New Import.

Non-English and Special Characters in Excel

If there are non-English or special characters in your posts, Microsoft Excel may save the CSV file using text encoding that is incompatible with the Message Importer. To avoid this, we recommend that you use another spreadsheet application (such as Google Docs spreadsheet) or a text editor, such as Notepad to create your CSV file. If you want to use Google Docs along with the Message Importer template, do the following:

  1. Download the template from to your desktop, but do not open it.
  2. From within Google, click Documents.
  3. Select Create > Spreadsheet..
  4. From the (Untitled) spreadsheet, select File > Import, click Browse and select the template.
  5. Click Import (Import Action: Create New Spreadsheet, Separator character: Comma)
  6. Click Open now.
  7. Modify the formatting for the Date and Time columns. (Select a column, then choose Format > Number > Plain text). Ensure the time is formatted properly (HH:MM AM/PM). Note that Google Docs may convert the time to 24-hour time immediately upon importing the file.
  8. Complete the spreadsheet.
  9. Save the file to your desktop. Select File > Download as > CSV (current sheet). Do not use Excel to open the spreadsheet.
