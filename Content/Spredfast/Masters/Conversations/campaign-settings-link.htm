Link Tagging

General link tracking is configured when you integrate your Google Analytics or Omniture accounts in . Use these Link Tagging screens to create custom link tags for special purposes, or to integrate with Coremetrics or other analysis software. You should be familiar with the general terms and concepts of link tagging before creating custom tags. For assistance with custom link tag creation, contact Support.

Create Tag Templates

Create a Tag Template before creating specific tags.

  1. From Initiative Settings > Link Tagging, click Add Template.
  2. In the Add Tag Template screen, complete the following:
    • Name—Name for the template.
    • Variable—The variable supported by your service that you want to use, such as utp_campaign or utm_term.
    • Value—Leave this field blank; it will be populated later when you assign values to the tag.
  1. In the Variable Parameters section, Separator menu: According to the syntax used by your analytics service, select the character that acts as a value separator.
  2. If you need to add a parameter, click Add Parameter. In the Add Parameter screen, provide:
    • Name—A name for the parameter.
    • Values—Select from:
    • Static List—If you want to define a specific list of values, provide list items, separated by commas.
    • Auto from Channel—If you want to use different values based on social site, enter them here.
    • Auto from Account Set—If you want to use different values based on the Account Set used to publish the activity, identify them here.
    • Auto from Account—If you want to use different values based on the account associated with the Initiative, enter them here.
    • Auto from Labels—If you want to use all labels that have been applied to the message before publishing time. Select a Separator from the menu that will be used in between labels. Note that all of the labels will be visible in the link tag.
    • Unique ID—If you want the system to assign a unique value to each post that contains the link.
  1. Click Add Parameter.

Tags

Create a tag template before creating tags.

Create a Custom Link Tag

  1. From Initiative Settings > Link Tagging > select Add Tag.
  2. In the Add Tag screen, Type menu, choose either Standard or a saved Tag Template.
  3. Depending on the template type, provide the variable and value parameters.
  4. If you want to use this tag consistently in your links, click Use by default, and the tag appears as the “selected” tag when you choose tags to apply to a link.
  5. Click Add Tag.

Add a CMP Tag

  1. From Settings > Link Tagging > Tags, click Add.
  2. In the Name field, select Standard.
  3. In the Variable field, enter CMP.
  4. In the Value field, enter 101AR&ct=101AR12W&cr=stgsocial&csr=power&cm=h&ccy=us&cpb=stg&S_TACT=101AR12W*
  5. Click Use by default.
  6. Click Save.

Domains to Tag

Automatically ask users if they want to append a tag whenever activity is posted to the domains listed in this section, such as all your business domains.

  1. To identify domains for which users are asked if they want to add a tag:
  2. From Initiative Settings > Link Tagging, select Add Domain.
  3. Enter the URL. It doesn’t have to be well-formed; you can use example.com instead of http://www.example.com.
  4. Click Add Domain.

Import Custom Tags Created in Another Iniative

  1. From Initiative Settings > Link Tagging > Import Tags.
  2. Select the Initiative from which to import tags.
  3. If the selected (source) Initiative uses tag templates that include Account Sets as variables, provide variables for this Initiative’s Account Sets.
  4. Click Import.
