Facebook Page Post Targeting

For Facebook Pages only—Page Post Targeting allows you to direct messages to specific types of viewers. There are two types of Facebook targeting: Gated and Enhanced. The type of target you choose determines where the message is published, and who can see it.

  • Gated targets—allow you to post messages to fans based on location and language. Only the targeted fan audience will see the post, in the page’s Timeline and in the fan’s news feed.
  • Enhanced targets—allow you to choose additional criteria, including age, gender, education, and others. Messages posted using Enhanced targeting appear in news feed of the targeted fans, as well as to all other viewers of the page Timeline—whether or not the viewers are fans of the page, and whether or not they meet the targeting criteria.

Gated Targets

Facebook messages that use a Gated target appear only to fans (people who have liked the page) in specified countries, states/provinces, cities, or who use a particular language. Places/languages available for targeting are those supported by Facebook. Only one target can be chosen for each message, even if the message is going to multiple pages. (The targeting is applied to all Pages you select in Accounts).

For locations—You must start with a country. Two “levels” are permitted. If you select multiple countries, then “the world” is considered as the top level, and you can only select countries. If you select a single country, then that country is the top level, and you can choose to designate:

  • states and provinces within the country
  • cities in states and provinces, or
  • everybody in the country.

For languages—the message will appear to people who chose that language in their profile when signing up for Facebook.

Add a Facebook Gated Target to a Message

  1. Click Targeting. If you want to use a target profile that has already been created, choose it from the Select a Saved Profile drop-down menu.

User Restrictions May Apply
If a Location/Language description appear when you click Page Targeting, but you have no selection options, it means that your administrator has restricted you to publish Facebook messages only to that location/language.

  1. From the Targeting drop-down menu, select Gated.
  2. In the Location field, start typing the name of a country; will auto-complete the location using ones available in Facebook. Click on the country to add it.
  3. You can specify states/provinces, or cities to target.
  4. In the Language field, start typing the name of a language; will auto-complete the language using ones available in Facebook. Click on the language to select it.
  5. (Admin only) If you want to reuse this set of choices as a profile, click Save Profile As, provide a name, and then click Save.

When a message using a profile is copied, the targeting profile is copied as well, so long as the user who copies it is allowed to use the profile.

Once targeting has been added to a message, you can access it again by clicking the arrow in the Targeting section of the message composition page.

Enhanced Targeting

Facebook messages that use an Enhanced target appear in the news feeds of selected fans, as well as to all viewers of the page’s Timeline. Only one target can be chosen for each message, even if the message is going to multiple pages. (The targeting is applied to all Pages you select in Accounts).

Enhanced targeting can use the criteria listed below. The categories/subcategories that appear in the drop-down menus are provided by Facebook.

  • Location—You must start with a country. Two “levels” are permitted. If you select multiple countries, then “the world” is considered as the top level, and you can only select countries (25 max). If you select a single country, then that country is the top level, and you can choose to designate one of the following:
    • states and provinces within the country
    • cities in states and provinces
    • everybody in the country
  • Language—the message will appear to people who chose that language in their profile when signing up for Facebook.
  • Age—A range of ages, or a specific age (for example, 13-65).
  • Gender—All, Men, Women.
  • Interested In—Men, Women, Not specified.
  • Relationship status—All, Single, In a relationship, Married, Engaged, Not specified
  • Education—Anyone, in High School, in College, College Grad...subcategories (College name, major, specific or all graduation years after 1950 (200 max).
  • Workplace—List of companies known to Facebook (200 max).

Add a Facebook Enhanced Target to a Message

  1. Click Targeting. If you want to use a target profile that has already been created, choose it from the Select a Saved Profile drop-down menu.

User Restrictions May Apply
If targeting descriptions appear when you click Page Targeting, but you have no selection options, it means that your administrator has restricted you to publish Facebook messages only to a specific targeting profile.

  1. From the Targeting drop-down menu, select Enhanced.
  2. In the Location field, start typing the name of a country; will auto-complete the location using ones available in Facebook. Click on the country to add it.
    You can specify states/provinces, or cities to target.
  3. In the Language field, start typing the name of a language; will auto-complete the language using ones available in Facebook. Click on the language to select it.
  4. In the Age field, select a range or specific age.
  5. In the Gender field, select Men, Women, or All.
  6. In the Interested in field, select Men, Women, or All.
  7. In the Relationship status field, click the ON button to display the menu, then select up to four status criteria.
  8. In the Education field select a level. If you choose College, you can specify schools; begin to type a school name in the field; the field auto completes. You can also specify major and graduation year(s).
  9. In the Workplace field, begin to type a company name. If the company is known to Facebook, it auto-completes.
  10. (Admin only) If you want to reuse this set of choices as a profile, click Save Profile As, provide a name, and then click Save.

When a message using a profile is copied, the targeting profile is copied as well, so long as the user who copies it is allowed to use the profile.

Once targeting has been added to a message, you can access it again by clicking the arrow in the Targeting section of the message composition page.

Create Facebook Unpublished Page Posts

Unpublished Page Posts (sometimes called “dark posts”) are special Facebook messages that appear only as promoted messages in the Newsfeed. They never appear in the Page’s Timeline, and are often used by Brands to test the effectiveness of their messages, or to compare performance of different message texts without creating multiple posts on the same topic, or publishing too many similar posts at or around the same time. These posts can be created in for any authenticated Facebook Page. Once created, they can be published using Facebook Power Editor. To obtain metrics information for Unpublished Page Posts, see Company-wide Analytics Filters.

  1. While creating a Facebook message, in the Accounts pane, select a Facebook Page account.
  2. Click Targeting, and add gated or enhanced targeting if appropriate.
  3. Click the Don’t show this Post to my Organic Audience (Unpublished Page Post) checkbox.

Target a Twitter Message

You can target an organic tweet to users in one or more countries. Here are Some things to know about Twitter targeting:

  • The tweets will appear in Followers’ Timelines if Twitter determines the Followers are in the targeted country. The tweets will also appear in search streams created by non-Followers in the targeted country.
  • The tweets will not appear in the publishing account’s native Twitter timeline, unless the account is located in the targeted country. The tweets will appear in the Inbox, however.
  • Targeting is automatically applied to reposts, but is not applied to automated retweets.
  1. While creating a Twitter message, in the Accounts panel, select a Twitter account.
  2. Click Targeting.
  3. In the Location field, start typing the name of a country; select it when it appears. You can enter an partial term (such as King for United Kingdom) to find the country.

Target a Google+ Circle

For Google+ Company Page updates only. By default, Google+ posts are visible to any Google+ user who visits the Company page. With targeting, you can make posts visible only to members of specific circles. A Google+ targeted message can be sent to multiple circles for each account.

  1. While creating a Google+ message, in the Accounts menu, select a Google+ Company Page Account.
  2. Click Targeting. A list of the circles associated with the account appears.

You can use circle targeting on multiple accounts for the same Google+ message. To add circles to other accounts, or to edit the current ones, click Circle Targeting. Make the changes, and click Apply.

Target a LinkedIn Company

For LinkedIn Company Updates only. By default, LinkedIn Company updates appear to Linkedin users who are following the company. Messages can be targeted to followers whose current LinkedIn profiles include one or more of the following criteria: company size, industry, job function, seniority, geography.

Minimum Number of Followers Required
A LinkedIn Company targeted post must have an estimated reach of at least 100 followers. When a LinkedIn Company is authenticated in , the system verifies that the company has at least 100 followers. If it does not, the targeting feature is disabled for the company. The feature is subsequently enabled when the follower count reaches 100. When building targeted posts for multiple companies, if one of the selected companies has fewer than 100 followers, targeting is not permitted for any of the companies.

  1. While creating a Linkedin message, in the Accounts panel, select a LinkedIn Company Account.
  2. Click Targeting.
  3. Select targeting criteria using the drop-down menus. Note that the categories and sub-categories within each criterion are provided and maintained by LinkedIn. The maximum number of sub-categories you can choose in each type is as follows:
  • Company size—maximum of 10
  • Industry—maximum of 100
  • Job function—maximum of 10
  • Seniority—maximum of 10
  • Geography—maximum of 10
  1. Click Apply. A summary of the selected targeting appears.
  2. Click Edit, to change the target, or Close, to accept it.
