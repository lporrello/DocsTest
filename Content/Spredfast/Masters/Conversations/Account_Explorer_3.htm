Account Explorer

The LinkedIn Account Explorer provides Follower/Connection counts, by day.

LinkedIn Account Explorer Columns

Column

Definition

Initiative Name

The name of the Initiative containing the account

Account Set

The Account Set containing the account

Account

The account receiving the “Follow” or “Connect” from a reader.

Metric

The type of data provided; Followers (Companies) or Connections (Individuals)

Date, in format M/DD/YY

For this day, the Total number of Followers or Connections the account has.
