Stream Library

The Stream Library lists all the streams and Collections that are available to a user. It is divided into two sections: Active Streams and All Streams. The Active Streams list is organized by tab name and contains the streams that have already been added to tabs. The All Streams list includes all the streams created by all users in the Initiative, regardless of whether they have been added to a tab or not.

You can include the same stream in multiple tabs, though the refresh rate, number of visible items, and your current location within the stream vary depending on whether the tab containing the stream is in view. All instances of the stream will be updated in real time when a user takes action on an item in the stream. If a shared bookmark has been added to an item in the stream, the bookmark is maintained and updated in all instances of the stream.

You can also include multiple instances of a stream in the same tab (for example, if you want to see multiple versions of the same stream with different filters applied to each one).

See Stream Names and Aliases for information about how to manage the appearance of multiple instances of a stream in a tab.

Add a Stream to a Tab

Activate the tab by clicking on the Tab Header.

Find the stream in the Stream Library. Either:

  • Click the + icon
    or
  • Drag the stream into the active tab.

Remove a Stream from a Tab

  1. Locate the stream in the tab’s section of the Stream Library.
  2. Click the - icon. Note that the stream remains in the All Streams section of the Stream Library.

Delete a Stream from the Inbox

  1. Locate the stream in the Stream Library.
  2. Click the Delete (trash can) icon. If others are using the stream, a warning is provided. Click Delete.

Edit a (Search) Stream

When you edit a search stream, all items found using the previous search definition are deleted. For more information about creating search streams, see Search Streams.

  1. Locate the stream in the Stream Library.
  2. Click the pen icon (Edit).
  3. Change the search terms, or add channel-specific options.
  4. Click Save.

Reorder Streams in a Tab

  • In the Tab view, grab the stream by its handle in the upper left-hand corner, and drag it to a new location.
    or
  • In the Stream Library, locate the stream within its active tab, and drag it to a new place within the tab’s list.
