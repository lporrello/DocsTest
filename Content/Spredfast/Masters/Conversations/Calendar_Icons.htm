Message Status Icons

Message Status Icons

Icon

Description

Needs Content. The item has been created and assigned, but it lacks any content.

Draft. For microblogs: the message has been created, and may or may not have content, but has not yet been moved into the publish/approval workflow. For all other message types: the assignee has added content and saved a draft, but it isn’t yet ready for approval or review.

Ready for review. A Writer (with no publishing privilege) has created an item and there is no Writer Approval Team for the Initiative. The system does not alert anyone. Reviewers must search the Agenda for items in this state so they can review them, or run a filter to find the items. (Calendar > Basic Filter > Action status > Ready for review).

Pending Approval. The item is in the approval process, which may involve a single approval team, or multiple teams that review the item in a prescribed sequence. An item enters this status if

the item uses an Account Set that requires approval

a Writer submitted the content and there is Writer Approval Team

supplemental approval teams were added to the item when it was created

Rejected. A member of an approval team has deemed that the item does not meet approval criteria and is returning it to the author for further work.

Pending Publication. The item has been approved (if approval was necessary) and is in queue for publication.

Published. The item has been published to its destination(s).

Partially Published. The item contained multiple destinations and at least one failed initially. If the user chooses to ignore the error, the item is put in this state. Click the icon to open the Action page and retry.

Failed to Publish. The item failed to publish, or the item had multiple destinations and one or more of them failed. The item is put in the Failed to Publish state until a user chooses to “Ignore error.” Once Ignore Error is selected, the item is put in the Partially Published state.

The item is an Unpublished Page Post (“dark post”)

The item is a Previously Unpublished Page Post (created as an Unpublished Page Post, then subsequently published).

Content Promotion Icons

Content Promotion Icons

Icon

Description

Pending—The promotion has either not yet been sent to Facebook, or is waiting for Facebook approval before it can be published.
Paused—The promotion is not running in the Newsfeed; it has been stopped temporarily. No money is being spent, but the end date is still in effect.
Pending Facebook Billing —The promotion is pending; log in to Facebook and update the billing/account information.

Active—The promotion is running in the Newsfeed. Status metrics are updated once per hour.
Complete—The promotion is not running; the duration period for the promotion has expired.

Failed—The promotion could not be published to Facebook. Users can try to resubmit.
Promotion error—Failed to communicate with Facebook.
