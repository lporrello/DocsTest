Add a Non-Publishing Calendar Entry

In the Week view, users with sufficient permissions can add non-publishing calendar entries based on Custom Channel type.

This feature enables you to add non-publishing content and events to the calendar so that you can view them along side your scheduled social content.

Custom Channels are flexible and customizable. If your administrator has enabled this feature, you can create calendar entires based on custom channels.

Content added to the calendar can include things like off-line events, email, web, non-supported social channels, and images. To learn more, visit the customer community blog.

Follow the steps below to add a custom channel calendar entry:

  1. Ensure that the Week view is selected for the Calendar.
  2. To the far right of the window, click Add, and then choose a Channel from the drop down that appears. The Add [Channel] Entry dialog box will appear.
  3. Enter a title, descriptive text, and an image (if you want to); then, set a schedule and add labels. Labels is the only field that is not required.
  4. Click Add. Your Channel item will appear on the calendar along with the icon that is defined for the given channel.

Editing and Deleting
If you want to edit a Custom Channel entry that is already added to the calendar, hover over the entry, and then select the pencil icon from the from the pop-up that appears. To delete an entry, select the trash can.
