Workflow

Use the Workflow pane to change assignees, add supplemental approval teams, and add notes to a message.

Assignment

To change the assignee for a post, click in the Assigned to menu and select a user.

If you don’t have permission to assign messages, only your name appears in the list. If you have permission to assign messages, then the list includes all users who have permission to Create and Edit Own Messages, and have a workspace that includes both Publishing and the Calendar (set to Yes).

Approvals

For messages that require approval before they are published, Approvals shows the required teams (based on the selected Account Set(s)), in the order they will be notified. You can require approval from supplemental (extra) teams; available ones appear in the list. (For a team to be available, at least one member of the team must have access to the Initiative.) If you make any changes to the set of approvals, the approval path is reset, and the message has to receive all required approvals again, even if the message content has not changed.

For more information on the approval process, see About Approvals.

Making Changes to Supplemental Approval Teams

  • For individual microblog messages, you can make changes to the list of supplemental teams at any time.
  • For microblog messages that are part of a Series, you can make changes to the list of supplemental approval teams, but you must make the change on each individual message in the series; making a change on one does not make the change on the rest of the messages in the series.
  • For individual messages of other types (video, photo, presentation, discussion topic), you can make changes to the list of supplemental teams up until content for the message has been saved. Once you save content, you can no longer make changes to the list of supplemental teams, even if you change the schedule for the message.
  • Similarly, for other message types (video, photo, discussion topic), that are on a repeating schedule (the Repeat/Repeat every fields are set), you can make changes to the list of supplemental teams up until the content for the original item is saved. Once content for the first item is saved, changes cannot be made to the list of supplemental teams for any of the following “repeats” of the message.
