Search Streams

Search streams allow you to find content on social media channels based on simple or complex terms or phrases. Search streams are the only type of stream that can be edited.

  1. From the Stream Library menu, select Create Streams.
  2. From the drop-down menu, select Search-Based.
  3. Enter terms, separated by commas, in the Search for all these words field. Terms are not case-sensitive. Do not use wildcards or special characters.
  4. To create a more complex search, click the Advanced Search checkbox. Enter search terms separated by commas. Search types are described in the table below. Not all channels support advanced search functions; if the channel does not support the search criteria, the checkbox for the channel is disabled. You may be able to work around the issue, though, by using a wider search, and then filtering the results within the stream.
    Facebook searches are limited to two keywords. Facebook search always interprets an “AND” operator between keywords; if you want an “OR” search, you must create a separate stream for each keyword. If you want to monitor multiple terms, you might want to create a separate stream for each, then put them into a Collection.
  5. Select channels to search. When you choose multiple channels, a separate stream is created for each channel. Some channels provide additional field parameters. Click the Supplemental Channel Search checkbox, and complete the fields per channel as described below:
  • Facebook—Choose Facebook accounts whose friends’ accounts you want to include in the search.
  • Flickr—Specify photos, screenshots, other items, or a combination of these. Limit the content to a specific User ID.
  • Social Mention—Select a category (blogs, microblogs, comments, and the like) and specify a Location (City and State/Province).
  • Twitter—Limit the search to a specific hashtag, or specify a Location (address) and distance from it to search. You must click Apply location for the system to include the place. When you create a Twitter search stream, returns results both from the content of the tweet, as well as the metadata for the tweet, such as the username, the application from which it was posted, and the URL. When looking at stream items, it may appear that there are no search terms present; the item is likely part of the tweet’s metadata.
  1. Each Twitter search-based stream must be associated with an authenticated Twitter account. By default, the stream is assigned to the account that has the fewest streams already associated with it. To change the assignment, click Change account and select it from the drop-down menu. Note that the number of search streams allowed for each Twitter account is limited, see Info and Twitter-Based Streams.
  2. Click Add.
  3. When you have added all the streams you want, click Done.
