Edit a Message

The pen icon (Edit) appears only before the message is published, and only to users who have appropriate permissions to edit the content and/or the schedule of a message. If the message requires approvals, and a user makes changes to the content, publishing accounts (Account Sets, credentials, retweet credentials), approvals list, or schedule before it is published, the message has to go through the entire approval process again. Users can make changes to the message title, assignee, targeting, labels, and notes without causing a re-start of the approval process.

To stop a microblog message (in Pending Publication status) before it publishes, click Edit and Save as Draft to return the message to Draft status. The message will have to go through the approval process again.

  • For (microblog) Messages: Click the pen icon (Edit). The same screen used to create the message appears, and you can perform all editing/scheduling tasks within it.
  • For all other Message Types: Hover over pen icon, and click either Edit Content or Edit Schedule.
