Google+

Comment

You can add a simple comment by clicking the Comment icon, typing in the content field, and clicking Comment. will use the account that originally created the stream to publish it. Or, click the More options button to use an expanded set of publishing features (other accounts, scheduling, notes, labels, etc.) for your post.

Delete from Google+

Google+ page administrators can delete posts and comments on posts. Deleted items are marked with a deleted (trash can) and added to the Hidden/Deleted stream. In situations when the items cannot be deleted from their native platform using , you have the option to mark them “deleted” using Delete on Spredfast, so that displays them in the Hidden/Deleted stream for tracking purposes.
