Export Calendar

Export items that appear on your calendar based on the filters you have set; up to 13 months of data can be included in a single export. Data is provided in a CSV file; once the file has been generated, a link to it is sent to the email address associated with your account. (Unlike some types of emails from , this one cannot be disabled.) Depending on the number of items contained in the export, it may take a few moments for the file to be created. You must be logged in to when you download the file using the email link.

The ability to Export Calendar data is controlled by a user’s role. See User Roles and Permissions

Message Importer vs. Export Calendar
You cannot export calendar data, then re-import it with the Message Importer; these features operate independently.

Export Data from the Calendar

Configure the calendar filters to include the appropriate Initiatives and other criteria.

  • Click the Export icon.
  • Use the calendars to set a date range.
  • Click Export.

For each message, the file contains the following data.

Calendar Export Data

Item

Definition

Initiative

The Initiative through which the item is published.

Channel

The channel through which the item is published.

Account Sets and Accounts

The Account Set(s) and Account(s) through which the item publishes, separated by commas if there are multiple, in the following format: Account Set: Account, Account Set2:Account, Account Set2:Account

Date

The date on which the message has published, or is scheduled to publish, in MM/DD/YYYY format.

Time

The time at which the message published, or is scheduled to publish, in HH:MM AM/PM format. Note that times shown in the Export file reflect the time zone of the user requesting the export.

Title

The -internal (not published) title of the message.

Content

The text of the message.

Labels

Labels that have been added to the message, separated by commas.

Notes

-internal notes on the message, separated by commas.

Status

Publishing status of the message.

Promoted

(Facebook only) If the message was Promoted on Facebook, this column reads Yes.

Promoted Budget

(Facebook only) The lifetime budget set for the promotion on the message.

Promoted Spend

(Facebook only) The amount spent to date on all promotions associated with the message.

Target Profile

(Facebook only) The name of the (saved) targeting profile used for the message.

Auto-imported

If the message was auto-imported into , this column reads Yes.

Reply

If the message was published as a response from this column reads Yes. (See the Basic/Advanced Filter for a definition of responses).

Assignee (email)

Email address of the user assigned to the message.

Approval path

Approval team next in line to approve the message; this field is blank for messages that have published;

Image Name

File name for an image attached to the message.

Published Image Link

If the message has been published, a link to the image on the channel.

Video Name

The title of the video (if one has been given) or the file name, if no title is present.

Log

Entries made about the message in the log, in the format of Action: User: Date Time, for example: Created: John Doe: 1/12/2013 10:46 AM. Note that times shown in the Export file reflect the time zone of the user requesting the export.

Link Preview Title

If available, the Link Preview Title (headline) is provided for microblogs

Link Preview Description

If available, the Link Preview Description is provided for microblogs

Permalink

If available, a permanent link to the message on the native channel. For Facebook and Twitter microblogs, Facebook photo albums and videos. Provides Account Set and account information, in this format: Account Set:Account:Permalink

Calendar Export data is obtained from a database replica, not the production databases; in some cases there may be discrepancies between production status and exported data. Any latency should be short (1-5 minutes).
