Activity Metrics

Activity metrics consist of data the system captures internally about user actions such as counts of posts, tweets, comments, photos, videos, and the like. It is updated and displayed every three hours.

If activity was auto-imported from an external source (such as Facebook or Twitter), its data is included in metrics at least twice a day. Activity metrics are imported at the same time that content and credential metrics are collected from a social channel, as described below.

Posting to Multiple Accounts

Within , you can publish an item to multiple accounts at the same time. For example, you can post the same message to three separate Facebook accounts. In calculating metrics, the system would consider this one Facebook post, not three.
