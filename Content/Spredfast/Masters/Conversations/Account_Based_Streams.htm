Account-Based Streams

Once you have authenticated accounts, set up streams to monitor activity on the social channels. The types of streams available are different for each channel.

  1. From the Stream Library menu, select Create Streams.
  2. Select a channel from the drop-down menu. For each channel, a list of available accounts appears.
  3. Click on an account name. The available stream types appear in the Add a stream to your Inbox screen. Select one.
  4. Click Add.
  5. When you have added all the streams you want, click Done.
Account-Based Stream Types

Stream Name

Definition

Bazaarvoice

Provides Bazaarvoice reviews, including Author name, review date/time, location, title, review text, star rating, review feedback level.

Brandwatch

Provides social media monitoring data from Brandwatch based on authenticated company and selected Project and Query. Stream items include metrics from Brandwatch: Tags, Gender, Language, Impact, Sentiment.

Crimson Hexagon

Select monitors from the list associated with your account. Each selected monitor receives its own stream and metrics. See Crimson Hexagon

Facebook Albums and Photos

Provides all albums, photos, and comments/replies posted to a designated Facebook Page, and presents them in chronological order.

Facebook Wall Posts

Provides all posts made to a designated Facebook Wall, and presents them in chronological order.

Facebook Page Activity

Provides all comments, replies, and posts made to a designated Facebook Page, and presents them in chronological order. When a new comment is made on a post, it appears at the top of the stream column, so that recent comments made on an older post can easily be monitored. Posts (and related comments and replies) are available for the past 100 days, which is a limit placed by Facebook.

Facebook Private Messages

Provides messages and photos that have been sent to a Facebook page using the Private Message feature from Facebook. Facebook page administrators can Reply to a Private Message from within the stream, but can only send two private messages without a response from the message author. After two messages have been sent without a response, the option to send a private message is disabled until a message from the author is received. Messages must follow any approval paths associated with the Account Set. There is a character limit of 500 in a response to a private message. Note that because of an API limitation in Facebook, non-photo attachments (such as videos or PDFs) are not supported in Private Messages.

Google+ Posts and Comments

Provides all posts and comments made to a designated Google+ Company page, in chronological order.

Instagram Timeline Activity

Provides all photos and videos posted by an account, plus those posted by people in the authenticated account’s list of followers, including comments they have received, in chronological order.

Instagram My Photos and Videos

Provides all photos and videos posted by an account, including comments they have received, in chronological order.

Instagram Hashtag Search

Provides all photos and videos that have the specified (single) hashtag applied to them, in chronological order. If you want to monitor multiple hashtags, create a stream for each one, then put the streams into a Collection.

LinkedIn Company Activity

Provides updates to LinkedIn Company activity, of the following types: status updates, comments on updates, job postings, new hires, new products.

LinkedIn Discussions and Comments

Provides all posts made to a LinkedIn discussion Group. Comments on the posts are provided if the account posted the topic, interacted with the topic, or clicked the star icon for the post from within . For more information, see LinkedIn Group Streams.

LinkedIn Updates

Provides all updates made by or about the people in this LinkedIn account’s network.

Weibo Weibos

Provides all Weibos (posts), comments, and replies in chronological order. When a new comment is made on a post, it appears at the top of the stream column, so that recent comments made on an older post can easily be monitored.

Twitter All Friends

Provides all tweets made by the people (Twitter accounts) that this account is following

Twitter Mentions

Provides all mentions and retweets of this Twitter account by others.

Twitter Your Tweets, Retweeted

Provides all instances in which tweets from this account were retweeted by others

Twitter Direct Messages:

Provides all Direct Messages sent to the Twitter account.

Twitter Lists

If the Twitter account has configured lists, they are displayed as available streams. The stream provides all tweets made by individuals (Twitter accounts) who are members of the list.

YouTube Video Comments

Provides comments made by others on videos posted by the YouTube account

Wordpress

Provides comments made on Wordpress blogs
