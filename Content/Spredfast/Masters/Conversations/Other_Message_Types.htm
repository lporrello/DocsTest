Other Message Types

For all non-microblog messages, click the ellipsis next to the channel icons on the Publish menu. The Schedule Message screen allows you to create and schedule all other types of messages. For information about microblog Series, which can also be configured from this screen, see Microblog Series.

Create Other Message Types (Non-Microblogs)

  1. From the > Publish menu, click the circled ... (ellipsis) icon to the right of the publishing icons.The Schedule Message screen appears.
  2. In the Name field, give the series a name (the name is internal-only, and will not be published).
  3. In the Type field, select a type of message to post.
  4. (For Blog, LinkedIn Discussion topic only) Click Allow immediate publishing if you want the message to post as soon as all required approvals are received.
  5. In the Date field, use the calendar to select the date on which the first message in the series will publish.
  6. In the Time field, use the picker to set the time the messages will publish (within a five-minute window).
  7. In the Repeat field, choose the increment at which the item (with the same content) will repeat: Daily (by the day), Weekly (by the week) Monthly by Day of Month (for example, May 9, June 9, July 9) or Monthly by Day of Week (for example, Second Wednesday in May, Second Wednesday in June, Second Wednesday in July, based on the characteristics of the date chosen in the Date field).
  8. In the Repeat every drop-down, select the frequency at which the message should repeat.
  9. In the End Date field, use the calendar to choose the date to stop the series.
  10. In the Labels field, add labels to the posts.
  11. In the Assign to field, select the assignee.
  12. In the Add approval path field, If you want to use supplemental approval paths for the message (in addition to the ones required for the Account Set) select them from the list.
  13. Click Save (to put the message on the calendar and add content later) or Save and Add Content to continue building the message by adding content and Account Sets/accounts.
  14. Once the message has been created, you can click View in Agenda to see it on the Calendar.

The fields required in the Action: [Message Type] screen content fields vary according to type of message; see the tables in the sections below.

Account Selection for “Other” Message Types
The first time a user publishes to a given channel, no account is pre-selected from her available accounts. After the first time, the system pre-selects the last set of accounts used.
