Process Individual Stream Items

You can process an individual stream item using the icons in the Actions bar, or the More actions menu—these appear when you move your cursor into the item. Menu options available for an item vary according to the stream in which it is viewed.

Depending on the stream type, you can...

  • add labels (that are available within the active Initiative)
  • highlight the item for another user (the user’s workspace must allow access to the Inbox)
  • apply a macro to an item
  • add, move, or delete a shared bookmark (within all Shared Collections and all streams except the Highlighted stream).
  • add a note
  • add the author to the list of influencers
  • amplify the item by publishing it to another social channel
  • see details about the author
  • assign a Crimson Hexagon category—the category is also used as a label in .
  • view the item in greater detail
  • translate a non-English item into English using Google Translate, and return it to its original language.
  • mark the item as “read”
  • archive the item
  • accept/release the item (Highlighted stream only)
