Video Trends

This table enables you to see aggregated video metrics visualizations over time.

The Video Trends tab includes the following tables:

  • Top Videos by View Over Time
  • Video Metrics Over Time—Hover over the metric that you are interested in the list of metrics at the bottom table. To see fewer metrics, filter using the Metric Set drop down control.
  • Video Metrics Table Over Time—This table provides an alternative view of the Video Metrics Over Time table.
