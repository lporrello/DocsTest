Label Messages

You must have sufficient permissions on all messages you select; if you select a message for which you don’t have permissions, the menu is disabled. For labeling, you must have permission to edit the message (if it hasn’t been published yet) or have had permission to edit the message (if the message has been published already). If you apply a label that is part of a label group (mutually exclusive labels), and the message already has one of the group labels applied to it, the label is changed to the new one. For example, if you apply the label “closed” to a message that has “open” already applied to it, the label “closed” replaces the label “open.”

Add Labels to Multiple Messages in the Agenda

  1. From the Calendar > Agenda tab, hover over the upper left-hand corner of an item until a checkbox appears; click it to select the item. To select all the day’s messages, click the checkbox to the left of the Details menu. The Add Labels button appears in the Agenda header.
  2. Click Add Labels.
  3. In the Add Labels screen, Add a label field, type the label(s). If the labels are not present in , they are created.
  4. Click Save.
