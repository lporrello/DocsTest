Blog Post

Blog Fields
Field Description
Title Name of the blog post
Post Content for the blog post; can include images (gif, png, jpg), embedded media, html.
Tags Descriptors for the post so readers can find it online.
Account Sets/Accounts Authenticated sites on which to post the blog.
Footer Include default footer text
Announce To announce on microblog channels, click the checkbox, write the announcement in the Template field, and click the checkbox for each channel to announce through.
Labels Choose labels to apply to the blog post within .
