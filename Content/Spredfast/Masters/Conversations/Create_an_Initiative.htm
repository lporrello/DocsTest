Create an Initiative

  1. Click in the Initiative Selector, to display the list of current Initiatives.
  2. At the bottom of the column (list), click Add Initiative.
  3. In the Name field, type a name.
  4. In the Description field, briefly explain the purpose of the Initiative.
  5. Click Create.
  6. Once the basic framework for the Initiative is in place, go through the Initiative Settings tabs (described in the Initiative Settings sections) to complete configuration.
