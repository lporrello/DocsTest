Stream Details View

By default items in the Inbox appear without details. You can control the appearance of detailed information for an item, for a stream, or for the entire Inbox. “Details” includes information such as Likes made via , Label, Note and Highlight counts, whether the item has been accepted by a c user (and if so, the name of the user) responses to the item, and access to comment information.

To always show details, go to the Stream Library menu and set the Stream Details Open setting to ON. To show/hide details on the item or stream level, use the Details icons.
