Publish an Asset

  • You can publish an asset directly from the Content Center if you have the following:
  • a role that permits publishing
  • access to a publishing account
  • folder-level publishing permission on the folder containing the asset.
  1. From within a folder or the All Assets view, click on an asset. A panel containing the asset and its properties appears on the right-hand side.
  2. Click Publish, and select a channel. Note that if the channel does not support the asset type (such as an image on LinkedIn), the channel appears in the list but cannot be selected. The publishing dialog appears.
  3. Make changes to the message or publishing properties. For more information on publishing, see Publish.
  4. Click Publish.
