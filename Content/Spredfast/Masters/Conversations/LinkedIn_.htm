LinkedIn Message Explorer

For details about filters on , see Company-wide Analytics Filters.

In the Message Explorers, the Total Number of Messages number is a drill-down; when you click it, additional columns of message-related data are provided.

LinkedIn Columns

Metric

Definition

Date

Date the message was published.

Time

Time the message was published, in Pacific Time.

Account

The account used to publish the message.

Labels

Any outbound label(s) applied to the message.

Image

Thumbnail of message image. Click on the link to open the image in a new window. (You must be logged in to the channel.)

Text

The content of the message. Click on the text link to access Message Details.

Channel ID

Unique ID for the message, created by the channel. Click the link to see the item on the native channel, in new window. (You must be logged in to the channel.)

Engagement

Sum of clicks, comments and likes the message received.

Clicks (Bitly or BudURL)

Number of clicks received by links embedded in the message, provided by Bitly or BudURL.

Comments

Number of comments the message received.

Likes

Number of likes the message received.

Web Conversions

If Google Analytics or Omniture has been configured for one or more of the Initiatives, the number of conversions that the message has generated.

Impressions

The number of impressions the message received (Company Accounts only)

LinkedIn Clicks

The sum of all clicks on links within LinkedIn Profile and Company Updates, and Group Discussion topics published from that are part of the Initiative, as well as clicks on links in Company and Profile updates that are auto-imported. Note that these click counts are NOT included in engagement calculations in , because many of these links are also Bitly or BudURL links, which could result in counting a link twice.

Shares

Number of shares the message received (Company Accounts only)

 

Additional LinkedIn Columns, from Total Number

Column

Definition

ID

Internal message identifier, used to de-duplicate message data.

Initiative Name

The name of the Initiative in which the message was published.

Account Set

The Account Set used to publish the message.

User Name

The name of the () user who published the message. If the message was auto-imported, (Auto import) appears in the column.

Title

The (internal, non-published) title of the message.

# of Messages

Internal value, used for calculation purposes.
