Collections

A Collection stream combines several individual streams into a “super stream.” The following rules apply to Collections:

  • An individual stream can be a part of many users' Collections.
  • A user can have an unlimited number of Collections, and each Collection can contain an unlimited number of streams. However, the Primary Collection, Highlighted, All Activity, Recent Engagement and Deleted/Hidden streams cannot be placed into a Collection.
  • There are two types of Collections: Private and Shared. Private Collections are user-specific; only the user who creates them can see them or edit them. Shared Collections are created and managed by users whose Workspace assignment allows them to add streams; they can be seen by any user in the Initiative. Private Collections are represented by a folder icon; Shared Collections are represented by an icon of a folder with an ellipsis inside.
  • If a user is allowed to add streams, he can change his Private Collection into a Shared Collection.
  • Once a Shared Collection is made available, each user can decide whether to include it in a tab.
  • A Shared Collection can be edited, made private, or deleted by any other user with a Workspace that allows him to add streams.
  • Within the Stream Library you can see the relationship between streams and Collections. On each Collection icon, the number of streams is shown; roll over it to get a list of the streams the Collection contains.
  • Like items in the Primary Collection, items displayed in a collection can be seen in the collection for 90 days. To see items older than 90 days, you must look in the source (constituent) stream separately.

Create a Private Collection

  1. From the Stream Library menu, select Create Collection. The Create Collection screen appears.
  2. Enter a name for the Collection.
  3. Drag streams from the All Streams column into the target area in the Collection list to add them.
  4. Click Done. The Collection is created and appears in the All Streams section of the Stream Library.

Once a Collection is created, you can add it to tabs as you can with other streams. If you delete a Collection, the individual streams that comprised the Collection are not deleted; they remain available for use.

For information on creating Private and Shared Collections, refer to the User Guide.

Change a Private Collection to a Shared Collection

  1. In the Stream Library, locate the Private Collection
  2. Click Edit (pen icon).
  3. Click the Share checkbox (to remove the check) located beneath the Collection name.
  4. Click Done.

Create a Shared Collection

  1. From the Stream Library menu, select Create Collection. The Create Collection screen appears.
  2. Provide a name for the Shared Collection.
  3. Click the Share checkbox beneath the Collection’s name.
  4. Drag streams from the All Streams column into the target area of the Edit Collection screen to add them.
  5. Click Save.
