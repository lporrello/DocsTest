Pinterest

Publish a pin from a file, asset in the Content Center, or a URL to a Pinterest Board. When you provide a URL, embedly returns a set of images that comply with Pinterest specifications. filters these images by size (100p H x 200p W) and shows you the ones you can select from.

Publish a Pinterest Pin

  1. From the , select Publish.
  2. Click the Pinterest icon. (Icons appear if there is an authenticated account for the channel, and you have access to post on an account of that channel.)
  3. Click the Pin tab.
  4. In the Select Board drop-down menu, choose a Board. You can post to one Board at a time.
  5. In the Title field, provide a name for the message; the name is not published.
  6. To upload from your computer, click Computer. Select the image then enter a URL for the pin in the field.
  7. To use an asset from the Content Center, click Content Center. Only assets that fit the specifications for Pinterest appear.
  8. To upload from a URL, click Website. Type a URL. It does not have to be well-specified.Click Next. Images from the URL that are appropriate for Pinterest appear. Click on an image. It appears in the message area, similar to a link preview image. You can select among the other available images using the < and > buttons.
  9. In the Enter a description for pin field, provide a summary description for the pin. If default text for the image is provided by the site, it appears in the description field; you can edit this text by clicking the pen icon.
  10. To add labels to the message, click the Labels icon. See Compose.
  11. By default, messages (Pins) publish as soon as possible. If you want to specify a date and time to publish, click the Calendar icon. You can also use terms such as “Next Thursday” or “Wednesday the 3rd” and the platform applies the correct date. See Compose.
  12. Click the Workflow tab.
  13. To assign another user to the pin, use the Assigned to menu. See Workflow.
  14. To add supplemental approval path(s) to the pin (the ones required by the Account Set are used automatically), click Approvals. See Workflow.
  15. To add notes to the pin for other users of to read, click Notes. See Compose.
  16. Click Create Pin (to start the approval process), or Save Draft (to save and create the message).

Once the message has been created, you can Copy Message or View in Agenda to see it on the Calendar.
