Post to Facebook, Twitter, Google+, LinkedIn, YouTube, or Weibo

Post a (microblog) Message

  1. From the , select Publish.
  2. Click the appropriate channel icon. (Icons appear if there is an authenticated account for the channel, and you have access to post on an account of that channel.) The Update Status dialog appears.
  3. Ensure that the Compose tab is selected.
  4. In the Account drop-down menu, choose accounts on which to publish the message. See Account.
  5. In the Message Title field, provide a name for the message; the name is not published, and if left blank, provides a name using the first four non-link terms in the text of the message.
  6. For YouTube, enter a Video Title, Video Privacy Settings, and YouTube tags.
  7. In the text area, type the content for the post. If the channel has a character limit (such as 140 for Twitter or Weibo), the number of remaining characters is shown in the lower right-hand corner of the message area.
  8. If you type a link as part of the text, automatically tokenizes and shortens it. See Compose.
  9. For Facebook, Google+, and LinkedIn, you can include a link preview on an embedded link. See Link Preview (Facebook, Google+, LinkedIn).
  10. If you want to add a photo (Facebook, Twitter, Weibo, Google+ only.) or video (Facebook only), in the Media field, select either Computer (and choose an image or video from your computer) or Content Center. See Photo or Video.
  11. If you have added a video, you can select Customize Thumbnail to add a customized thumbnail comprising either a frame from the video or a file from your computer.
  12. (Facebook only) To enable a call to action for an added video, select the call to action that you want to use in the Call to Action pull down menu, and then add a call to action URL in the Call to Action Link URL field.
  13. By default, messages publish as soon as possible. If you want to specify a date and time to publish, in the Schedule field, click in the Calendar field, and select a date and time. You can also use terms such as “Next Thursday” or “Wednesday the 3rd” and the platform applies the correct date. See Schedule.
  14. To add labels to the message, in the Labels field, use the Add Labels drop-down. See Labels.
  15. To add notes to the message for other users of to read, in the Notes & Activity panel, enter text in the Add note field. See Notes & Activity.
  16. If you want to target the post to specific viewers, or if the message is a Facebook Unpublished Page Post (dark post), click the Targeting tab. Note, YouTube does not support targeting. See Targeting.
  17. (Facebook only) To promote a microblog message in the Page’s Newsfeed, click the Promotion tab, turn On Promotion, and fill in the values. See Promotion.

    Can I Promote Video?
    You cannot promote videos at this time.

  18. Click the Workflow tab.
  19. To assign another user to the message, use the Assigned to menu. See Assignment.
  20. To add supplemental approval path(s) to the message (the ones required by the Account Set are used automatically), click Approvals. See Approvals.
  21. Click Publish Now (to start the approval process), or Save Draft (to save and create the message).

Once the message has been created, you can Copy Message or View in Agenda to see it on the Calendar.
