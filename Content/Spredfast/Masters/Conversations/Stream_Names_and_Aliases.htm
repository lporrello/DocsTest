Stream Names and Aliases

By default, newly created streams are given names according to their types and characteristics (such as [Facebook Page Name: Page Activity, or Search: apples for a search stream using the term “apples”). The name of a stream can be changed permanently by renaming it, or temporarily, by giving it an “alias.”

Any user can give a stream an alias. An alias is a unique, temporary name for a stream (in a tab) that’s assigned by an individual user. It is seen only by the user who creates it, and is forgotten when the stream is removed from its active tab. (An aliased stream can be moved from one tab to another, however.) An alias helps a user distinguish between multiple instances of the same stream within a single tab (such as when different filters have been applied to each one, for example, and the user needs to tell them apart). To give a stream an alias, click on the tab (so that the stream is visible), click on the stream’s name in the header, and provide a new name. Once created, the alias appears in the stream’s header, and in the tab’s list of streams in the Stream Library.

To rename a stream is to provide a new, permanent, universal name for the stream that appears to all users in the Initiative (once they refresh the page). To rename a stream, find the stream in the All Streams section of the Stream Library, click on the name, and provide a new one. The new name appears in all locations, for all users, (unless a user has aliased the stream in a tab).

You can also rename a stream when it is first created (shown as “added” in the Added Streams section of the Add Streams dialog). Click the Edit stream name (pen) icon to the left of the stream name, then in the Rename field, provide a new name for the stream and click Save.

Sysomos or Synthesio Streams
When multiple streams are created, they populate with the name, "[Vendor name] Query Stream". To ensure that stream names are not identical, you can create an alias for each stream. To do this, click on the stream name in the Stream Library and then give the stream a unique name.
