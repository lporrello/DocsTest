Additional Analytics

Web Analytics

Web analytics consists of site performance/conversion data for a website or app supplied by Google Analytics or Ominture, if those services have been integrated with . It is updated every 24 hours. The metrics collected vary based on configuration; common ones are described in the table below.

Common External Analytics Metrics

Metric

Description

Service

Notes

Bounces

Number of single page visits, or visits in which person left the site from landing page. The Roll-up value is the sum across the date range. This data is provided by Google Analytics or Omniture.

Google Analytics or Omniture

 

“From Links” shown beneath any metric

Number of items that were sourced through an embedded link in published activity. Click any highlighted number to see the messages.

Google Analytics or Omniture

 

New Visits

Number of new visitors to site (browser sessions). The Roll-up value is the sum across the date range. This data is provided by Google Analytics or Omniture.

Google Analytics or Omniture

 

Visits

Number of visitors, including new and returning visitors, to the site. The Roll-up value is the sum across the date range. This data is provided by Google or Omniture.

Google Analytics or Omniture

 

Page Views

Number of page views (actual views, non-unique) the site received. The Roll-up value is the sum across the date range. This data is provided by Google Analytics or Omniture.

Google Analytics or Omniture

 

Goal:

“Goals” are configured in Google Analytics or Omniture, and describe desired user actions, such as “sign up” or “purchase.”

Google Analytics or Omniture

 

Goal Value

If dollar amounts are associated with goals within Google Analytics or Omniture, the sum value of user activity is reported. The Roll-up value is the sum across the date range. This data is provided by Google Analytics or Omniture.

Google Analytics or Omniture

 

Crimson Hexagon

Opinion data obtained through Crimson Hexagon is updated each time new data is fetched.

Crimson Hexagon metrics are custom for each account and category. To see the Crimson Hexagon dashboard via , select Analytics > Additional Analytics > Crimson Hexagon. Enter your Crimson Hexagon login credentials.

Crimson Hexagon, a chart is provided for each category you’ve added as a stream in the Inbox. Because all Crimson Hexagon metrics are custom-created, the results that appear are different for each category.
