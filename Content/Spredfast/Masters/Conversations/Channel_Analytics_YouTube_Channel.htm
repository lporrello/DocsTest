Channels

This section of the YouTube Performance Dashboard provides data for YouTube pages only. It provides metrics for Subscribers, Views, and Videos. It also features a Return on Activity table, that enables you to view additional channels, and a Channels by Views table.

  1. To view additional channels, select View all 4 Channels to the right of the Return on Activity table header. A new dialog box will appear. In the all channels dialog box, you can view data in a table, line graph, bar graph, or pie-chart.
  2. To the top-right of the all channel dialog box, select the table, line graph, bar graph, or pie-chart icon. Hover over a bar in the bar graph or slice of the pie in the pie-chart to see details.

This tab contains the following information:

  • Channel
  • # of Videos
  • Avg. Engagement
  • Avg. Video Views
  • Channel Subscribers
  • Value (graphs only)
