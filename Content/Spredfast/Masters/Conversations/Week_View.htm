Week View

The Week view provides a summary of the week’s activity, for all Initiatives chosen in the Basic and Advanced filters. (Items from the current active Initiative have the Initiative name in plain text. Items from other Initiatives have italicized names.) The number of (total) messages is shown beneath each day/date for each day column. Click on the number to open the agenda view for the given day.

You can toggle between the collapsed and expanded view using the Details ON/OFF in the upper left of the page. Use the Basic and Advanced filters to see messages of a particular type, status, assignee, scheduled, or published in a given week.

Week View: Details Off

  • Time (scheduled or published)
  • Channel
  • Message status icon
  • Attachment thumbnail if a photo, video, or album is attached to the message

Week View: Details On

  • Time (scheduled or published)
  • Channel
  • Message status icon
  • Attachment thumbnail if a photo, video, or album is attached to the message
  • Assignee name or auto-imported
  • Message Name, if one has been provided by the user during message creation, and the text of the Name does not identically match the initial content of the message.
  • First lines of message text (if a Message Name is displayed) or two lines of message text, if no Message Name is displayed.

Click on a post to open a flyout with more information.

Post Information Flyout

The post information flyout that appears when you click a message contains the following:

  • Post status (Draft, Needs Content, Pending Approval, etc.)
  • Selectable approval options (Approval buttons appear only if the post was created with an approval workflow.)
  • The account name to which the post is made
  • Message text summary
  • Image thumbnail that you can click for an enlarged view
  • Action icons, such as edit, copy, star, and delete, and if the message is part of a series or repost.
  • Account Sets/avatar
  • Assignee
  • Initiative name
  • Real-Time Trends metrics link and summary (if you have a Intelligence subscription). Click the link to view metrics in Intelligence.
  • Editable note fields, including Labels, Notes, and Activity
