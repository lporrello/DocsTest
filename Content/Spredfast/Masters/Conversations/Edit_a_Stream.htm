Edit a Stream

Only Search streams are editable. You can change search terms, but cannot change the channel using the Edit feature. Note that when you edit a search stream, all items found using the previous search definition are deleted.

  • Click the Stream Management (chevron) icon in the stream header and choose Edit Stream
    or
  • Locate the stream in the Stream Library, and then click the pen icon (Edit).
    1. In the Search for all these words: field, add or modify search terms.
    1. To create a more complex search, click the Advanced Search checkbox. Enter search terms separated by commas. Search types are described in Search Streams. Not all channels support advanced search functions; if the channel does not support the search criteria, the checkbox for the channel is disabled. You may be able to work around the issue, though, by using a wider search, and then filtering the results within the stream.
    1. Twitter only—Each Twitter search-based stream must be associated with an authenticated Twitter account. By default, the stream is assigned to the account that has the fewest streams already associated with it. To change the assignment, click Change account and select it from the drop-down menu. Note that the number of search streams allowed for each Twitter account is limited, see Info and Twitter-Based Streams.
    1. Click Save.
