Initiative Analytics

displays graphs of the most important metrics for an Initiative. Use it to get a quick overview of Initiative status. You can choose which metrics appear on the dashboard from over 30 predefined charts. Information on the dashboard is updated at least twice a day.

Select Charts to Appear on the Dashboard

  1. From the , click Manage Panels.
  2. To include a panel on the dashboard from the Available Panels column, click the + icon.
  3. To remove a panel from the Active Panels list, click the - icon.

Panel Selections Apply Globally
When you make changes to the dashboard, the changes appear to all users.

  1. Click Save Changes.

Many of the panels allow you to export the underlying data as a CSV or Excel file, or an image of the panel as a PDF. Click the chevron icon next to the title, click Download As button, and select a file type.

By default, data presented on the dashboard spans the last 30 days. To change the time interval, or apply other filters to the dashboard, use the Filters feature. When using dashboard filters, remember that...

Applying the filter recalculates data for all the panels in the dashboard, not just the panel you're viewing.

When building a filter, clicking a checkbox includes the checked item’s data, and excludes data for the unchecked items. For example, in the Content type criteria, checking “Blog” means that only data for blogs will be returned, and data for Microblogs, Photos, and Comments will not appear.

In general, you can modify any filter criteria for any panel. However, some metrics have filters built in to their calculations (Top 10 Mentions by Audience uses only Twitter as the Social channel, for example), so adjustments you make to a Dashboard filter might not impact the results (if you filtered for Facebook, for example, and looked at the Top 10 Mentions by Audience). In such cases, the panel descriptions tell you which filters you can effectively adjust.
