Facebook Message Explorer

For details about filters on , see Company-wide Analytics Filters.

In the Message Explorers, the Total Messages number is a drill-down; when you click it, additional columns of message-related data are provided.

Facebook Columns

Metric

Definition

Date

Date the message was published.

Time

Time the message was published, in Pacific Time.

Account

The account used to publish the message.

Labels

Any outbound label(s) applied to the message.

Title

The (internal, non-published) title of the message.

Channel ID

Unique ID for the message, created by the channel. Click the link to see the item on the native channel, in new window. (You must be logged in to the channel.)

Image

Thumbnail of message image. Click on the link to open the image in a new window.

Dark

The message is a “dark” post (Unpublished Page Post)

The item was created as a dark post, but subsequently published. (Previously Unpublished Page Post, or Previously Unpublished Post)

Text

The content of the message. Click on the text link to access Message Details.

Organic Impressions

The number of unique views of the message.

Paid Impressions

The number of unique views of the message from a sponsored Facebook product, such as a Page post ad or sponsored story.

Viral Impressions

The number of unique views of the message originating from a story published by a friend (such as like, comment, or share stories).

Engagement Rate

Engagement rate (percentage) is the number of people who clicked on, liked, commented on, or shared a post (post_engaged_users) divided by the number of unique people who saw it (post_impressions_unique)

Engaged Users

The number of unique people who have taken any action on the post.

Reach

The number of people who have seen the post organically

Paid Reach

The number of people who have seen the post as a result of a promotion

Storytellers

The number of people sharing stories about the message.

Engagement

Sum of Stories and Clicks

Clicks

Number of clicks received by links embedded in the message, provided by Bitly or BudURL (depending on which you have configured).

Like Stories (Timeline)

Number of stories created by Facebook when a person liked a message.

Comment Stories (Timeline)

Number of stories created by Facebook when a person commented on a message.

Share Stories (Timeline)

Number of stories created by Facebook when a person shared a message.

Other Stories

Number of other stories (such as redeeming an offer or answering a poll) created by Facebook

FB Link Clicks

Number of times an embedded web link in the message was clicked.

FB Photo Views

Number of times a photo in the message was viewed.

FB Video Plays

Number of times a video in the message was played.

FB Other Clicks

Number of clicks made anywhere in the message, but not on web links in the content (e.g., on poster’s profile, “more...” link, etc.).

Negative Feedback

Number of items of negative feedback (Hide, Hide all, Mark as Spam, Unlike, X button) received by the message.

Web Conversions

If Google Analytics or Omniture has been configured for one or more of the Initiatives, the number of conversions that the message has generated.

 

Facebook Additional Columns, (Drill-down from Total Number)

Column

Definition

ID

Internal message identifier, used to de-duplicate message data.

Initiative Name

The name of the Initiative in which the message was published.

Account Set

The Account Set used to publish the message.

User Name

The name of the () user who published the message. If the message was auto-imported, (Auto import) appears in the column.

Title

The (internal, non-published) title of the message.

Dark

The message is a “dark” post (Unpublished Page Post)

If the item was created as a dark post, but subsequently published.

Message

A unique identifier for each message, published through or auto-imported.

# of Messages

Internal value, used for calculation purposes.
